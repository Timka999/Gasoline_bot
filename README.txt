Это простейший tg-бот, считающий расход бензина и оценивающий стоимость поездки.
Он также выдаёт расстояния между городами, присутствующими в таблице Russia_cities.csv. 
Получив токен у @BotHelp_Fatherbot любой желающий может запустить его самостоятельно, для этого в корневой директории нужно создать файл TOKEN.txt и вставить туда токен.
Файл Hello.txt содержит приветственное сообщение команды /start.
Файл Bot_Menu.txt содержит команды, которые нужно отправить @BotHelp_Fatherbot, выбрав свой бот/Edit Bot/Edit Commands, чтобы создать удобное меню.

Сейчас (по состоянию на 08.10.24 бот развёрнут по адресу @GasCount_bot (https://t.me/GasCount_bot)).
